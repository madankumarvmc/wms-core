# SBX WMSLite

Lite WMS truck-loading module for steel coil dispatch — a StackBox product.

SAP sends a loading plan (truck + the coils to load). An operator at the bay
types the truck number, selects the truck, sees every coil to load, scans each
coil's QR with a Zebra/DataWedge handheld, and confirms it loaded — one coil at
a time, any order. Offline-capable PWA + back-office supervisor console.
